# 2026-05-04 学習ログ（Node.js標準入力・split・配列内の個数カウント）

## 今日やったこと

Node.jsで標準入力を受け取り、数列の中に指定された整数 `k` が何個あるかを数える問題を解いた。

扱ったコード：

```js
const fs = require("fs");

const input = fs.readFileSync(0, "utf-8").trim().split("\n");

const n = Number(input[0]);
const a = input[1].split(" ").map((x) => Number(x));
const k = Number(input[2]);

let num = 0;

for (let value of a) {
  if (value === k) {
    num++;
  }
}

console.log(num);
```

入力例：

```text
5
1 2 3 2 2
2
```

この場合、

- n = 5
- a = [1, 2, 3, 2, 2]
- k = 2

数列の中に 2 は3個あるので、出力は 3 になる。

## 学んだこと

### 1. 標準入力の読み取り

- `const fs = require("fs");`
  - Node.jsに最初から入っている fs を使うためのコード。
  - fs は File System の略で、今回はターミナルから入力された内容を読み取るために使った。

- `fs.readFileSync(0, "utf-8")`
  - 標準入力をすべて読み取る処理。

- `.trim()`
  - 前後の余計な空白や改行を消す処理。

- `.split("\n")`
  - 入力を改行ごとに分けて、配列にする処理。

入力が次のようなとき：

```text
5
1 2 3 2 2
2
```

`input` はこうなる：

```js
["5", "1 2 3 2 2", "2"];
```

そのため、

- `input[0]` // "5"
- `input[1]` // "1 2 3 2 2"
- `input[2]` // "2"

のように取り出せる。

### 2. 数列を配列にする処理

```js
const a = input[1].split(" ").map((x) => Number(x));
```

- `input[1]` は "1 2 3 2 2" という文字列。
- まず `.split(" ")` でスペースごとに分ける。

```js
["1", "2", "3", "2", "2"];
```

- その後 `.map((x) => Number(x))` で1つずつ数字に変える。

```js
[1, 2, 3, 2, 2];
```

つまり、2行目の文字列を「数字の配列」に変える処理。

### 3. k の個数を数える処理

```js
let num = 0;

for (let value of a) {
  if (value === k) {
    num++;
  }
}
```

- `num` は、k が何個あったかを数えるための変数。
- 最初はまだ見つけていないので 0。
- `for...of` で配列 `a` の中身を1つずつ取り出す。

例：

```js
a = [1, 2, 3, 2, 2];
```

このとき、

- 1回目 value = 1
- 2回目 value = 2
- 3回目 value = 3
- 4回目 value = 2
- 5回目 value = 2

`value === k` で、今見ている数字が k と同じか確認する。
同じなら `num++` で `num` を1増やす。

今回の場合、

- value = 1 → 違う → num = 0
- value = 2 → 同じ → num = 1
- value = 3 → 違う → num = 1
- value = 2 → 同じ → num = 2
- value = 2 → 同じ → num = 3

最後に、

```js
console.log(num);
```

で答えを出力する。

## 実行方法

VSコードのターミナルで、

```bash
node a.js
```

を実行する。そのあと入力を貼る。

```text
5
1 2 3 2 2
2
```

- Ctrl + Z
- Enter

を押すと結果が出る。

出力：

```text
3
```

## 今日のポイント

- `fs.readFileSync(0, "utf-8")` で標準入力を読む
- `.split("\n")` は改行ごとに分ける
- `.split(" ")` はスペースごとに分ける
- `map((x) => Number(x))` で文字列の配列を数字の配列に変える
- `for...of` で配列の中身を1つずつ見る
- `value === k` で探している数字と同じか判定する
- `num++` でカウントを1増やす

## つまずいた点

- 入力は最初から数字の配列ではなく、文字列として入ってくること
- `.split(" ")` だけではまだ文字列の配列なので、`Number` で数値に変換する必要があること
- `for...of` は値を見るループであり、位置そのものは直接取り出さないこと

## 次回やること

- `for` を使って、値ではなく位置を答える問題にも慣れる
- `break` を使って、最初に見つかった場所だけを答える問題を練習する
- 偶数判定や条件分岐を組み合わせた探索問題に進む

## まとめ

Node.jsで入力を読み取り、数列を数字の配列に変換し、`for...of` を使って1つずつ確認することで、指定された数字 `k` が何個あるかを数えることができた。
