---

### 学んだこと

#### 1. 入力された値はそのままでは計算できない

```js
const n = Number(input[0]);
```

`input[0]` は文字列なので、そのままでは数値の計算に使えない。

`Number()` を使って数値に変換してから、足し算・引き算・掛け算をする必要がある。

---

#### 2. 変数を使うと意味がわかりやすい

```js
const wt = 10;
const week = 7;
```

数字を直接書くよりも、変数にしておくと「何の数字か」がわかりやすい。

`wt` は1年前の体重、`week` は1週間の日数という意味をコードに持たせられる。

---

#### 3. 問題文を式に変える練習が大事

体重の増加量は、

```js
n - 10
```

N週間後の日数は、

```js
n * 7
```

このように、問題文をそのまま読むだけでなく、式に変換して考えることが大事だとわかった。
